# 雀番 Jokhan :基于麻将元素的类《小丑牌（Balatro）》卡牌游戏设计

核心沿用**Balatro 构筑+倍率叠加+回合闯关+卡组运营**的核心玩法，替换为完整麻将体系（牌张、番型、规则、术语），命名为**雀胡弈**，兼顾原作爽感与麻将传统文化。

## 一、核心框架对标（Balatro ↔ 雀胡弈）
| 小丑牌(Balatro) | 雀胡弈（麻将版） |
| ---- | ---- |
| 卡组：普通卡牌 | 手牌：**麻将牌（万/条/筒+字牌）** |
| 小丑牌（核心倍率/特效） | **役牌/宝牌/特殊雀符**（核心增伤、特效、倍率） |
| 道具/塔罗牌/奇观 | **麻将秘具、风水牌、局外命格**（一次性效果、全局buff） |
| 下注倍率、得分体系 | **番数叠加、胡牌得分、底分**（麻将经典计分逻辑） |
| 闯关轮次、盲选挑战 | **牌局关卡、庄位挑战、四境试炼**（递进难度） |
| 弃牌、重抽、组合出牌 | **吃/碰/杠/弃牌/摸牌**（麻将基础操作） |

## 二、基础设定
### 1. 牌库构成（全局可用牌）
沿用国标麻将144张基础牌，分四大类：
- **序数牌**：万子(1-9万)、条子(1-9条)、筒子(1-9筒)，每类4张
- **字牌**：风牌（东南西北）、箭牌（中发白），每类4张
- 无花牌/百搭，后期解锁**百搭雀牌**作为稀有卡

每局起始**13张手牌**，遵循麻将基础牌型规则：**4组面子（刻子/顺子）+ 1对将牌** 即为胡牌（对应原作“打出有效牌组得分”）。

### 2. 核心目标
闯关制，每一关为**一局麻将对局**，对手为AI雀士。
要求在**限定摸牌轮次**内，组合出指定**最低番数**的胡牌型；番数越高、额外特效越多，通关评级越高，解锁更强道具与雀符。

## 三、核心玩法模块（复刻Balatro乐趣点）
### （一）「雀符」= 小丑牌（核心养成&倍率核心）
**雀符**是本局常驻效果，可在闯关间隙商店购买、战利品抽取、合成升级，最多携带8枚，每枚提供**番数加成、牌型特效、规则改写**，对应小丑牌的倍率+特殊能力。
分四大派系，搭配组合产生联动效果：
1. **基础番符（新手刚需）**
    - 连顺符：顺子每组额外+1番
    - 刻子符：刻子/杠每组额外+2番
    - 将对符：对子将牌翻倍加分
2. **大牌特效符（高爆发）**
    - 清一色符：达成清一色时，全局番数×2
    - 字牌符：风牌/箭牌组成牌型，额外追加3番
    - 杠上开花符：开杠后摸牌，本局所有牌组倍率+50%
3. **功能干扰符（运营向）**
    - 巧吃符：无视牌序限制，自由吃牌凑顺子
    - 暗杠符：暗杠不暴露手牌，额外多摸1张牌
    - 弃变符：弃牌有概率转化为任意所需序数牌
4. **传说稀有符（终极联动，对标传奇小丑）**
    - 大四喜符：集齐四风刻子，全局番数×5
    - 十三幺符：可直接凑十三幺牌型，无视轮次限制
    - 海底捞月符：最后一轮摸牌胡牌，得分×10

> 玩法亮点：多雀符叠加，实现**番数指数级暴涨**，复刻Balatro堆倍率的爽感。

### （二）「秘具」= 塔罗/奇观/消耗道具
单局一次性道具，每局最多携带5个，闯关、商店获取，应急改变战局：
- **换牌骰**：随机替换3张手牌（重抽）
- **四暗令**：本局所有刻子强制变为暗刻，+3番
- **补杠令**：手牌可额外补一次杠，不限次数
- **避雷符**：本局点炮不扣分，仅自摸得分
- **天运牌**：开局直接摸满14张牌，优先凑胡牌型（天胡效果）

### （三）对局流程（单局完整流程）
1. **开局配牌**：随机13张手牌，选择本局携带的**雀符+秘具**，确定本局目标番数
2. **轮次行动**（每轮=摸1张牌 + 选择操作）
    - 摸牌：从牌山摸1张，手牌变为14张
    - 可选操作：
      ① 凑牌：**吃/碰/杠**，组成面子，压缩手牌、叠加番数
      ② 弃牌：打出无用牌，保留核心牌型
      ③ 用秘具：触发一次性道具效果
3. **胡牌结算**
    凑齐「4面子+1将牌」即可胡牌，统计**基础番数 + 雀符加成 + 特效加成**，总番数达标则通关。
    - 自摸：全体AI对手扣分，得分翻倍
    - 点炮：仅点炮目标扣分，基础得分
4. **战后奖励**：根据总番数评级，奖励新雀符、金币、秘具，进入下一关。

### （四）倍率&计分逻辑（对标Balatro得分）
公式：**最终得分 = 底分 × 总番数 × 雀符倍率 × 牌型倍率**
举例：
基础底分=100，基础番=8番，清一色符(×2)+刻子符(+2番)，总番数叠加后，得分会快速膨胀，和原作堆分体验一致。

## 四、关卡&养成体系（闯关循环）
### 1. 关卡梯度（递进难度）
- **初阶坊市**：新手关，最低番数要求低，牌型宽松，解锁基础雀符
- **四方雀馆**：引入AI对手干扰（AI会刻意卡你需要的牌），限制摸牌轮次
- **庄位试炼**：玩家坐庄，庄家底分翻倍，但对手强度提升
- **四境天阙**：高难关卡，强制限定牌型（如“必须做字牌胡牌”“仅限顺子”），解锁传说雀符
- **无尽雀局**：无尽模式，无限叠加番数与难度，冲分榜单（对标Balatro无尽模式）

### 2. 全局养成
- 商店：用对局金币购买新雀符、升级旧雀符、补充秘具
- 牌库扩容：解锁稀有牌、百搭牌、专属限定牌
- 命格解锁：全局永久buff（如“每局起始多1张牌”“基础底分永久+20%”）

## 五、特色原创机制（融合麻将+原作创意）
### 1. 「宝牌系统」（强化堆倍率）
每局随机刷新**3张宝牌**，手牌中每有1张宝牌，**全局番数+1**；杠牌可追加宝牌数量。
后期解锁雀符可让宝牌效果翻倍，成为高分局核心构筑。

### 2. 「役种连锁」（组合构筑乐趣）
不同麻将番型（役种）形成连锁加成：
例：断幺九 + 平和 同时达成，触发连锁，额外+4番；
鼓励玩家主动搭配多重番型，而非单一大牌，丰富构筑思路。

### 3. 「对手博弈」（区别纯单机卡牌）
AI雀士有不同性格：
- 保守型：优先做小牌，频繁碰牌卡手牌
- 激进型：强行做大牌（清一色/大四喜），后期会截胡你的牌
玩家不仅要组卡组，还要预判对手，增加策略深度。

## 六、视觉&音效风格（国风氛围）
1. **画面**：水墨国风为主，牌桌采用中式红木八仙桌，麻将牌沿用传统白底青花/红漆样式；雀符做成**玉佩、符咒、牌匾**样式，特效用烟火、金纹、祥云。
2. **音效**：保留经典麻将音效（摸牌、碰、杠、胡牌），搭配国风民乐（二胡、古筝）；高番胡牌时响起喜庆锣鼓。
3. **UI**：字体用书法字体，关卡名称、提示语使用传统麻将术语，沉浸感拉满。

## 七、简化版（手游/休闲向）适配方案
如果做轻量化休闲版，删减复杂国标规则：
1. 移除吃碰的复杂博弈，**只保留手牌组型**，不用对抗AI对手
2. 简化牌型：仅保留顺子、刻子、对子三大基础牌型
3. 雀符效果简化，主打“无脑堆番、爽快得分”，偏向纯休闲刷分，完全对标Balatro轻度玩法

## 延伸创意（玩法彩蛋&联动）
1. **地域流派皮肤**：解锁川麻、广麻、沪麻风格牌桌&牌张，附带对应地域特色番型
2. **趣味挑战局**：限时关卡“只能做十三幺”“全程不能杠牌”等趣味约束玩法
3. **合成系统**：3枚低级雀符可合成1枚高级雀符，复刻卡牌合成乐趣

这套设计完整保留了《小丑牌》**卡组构筑、倍率堆叠、闯关刷分、道具搭配**的核心乐趣，同时深度绑定麻将文化，玩法可深可浅，既能做硬核策略单机，也能改成休闲手游。
